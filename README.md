# COMAM Portal AI

## Español

COMAM Portal AI es una plataforma web institucional para la Conferencia Masónica Americana — COMAM. El proyecto contempla un portal público, gestión de contenidos, historia institucional, artículos, documentos, estatutos, logias y obediencias participantes, además de un módulo de registro para la Conferencia COMAM 2026 en Santiago de Chile.

La solución incluye un backoffice administrativo seguro con autenticación de doble factor mediante OTP, administración de usuarios, gestión de registros, invitados, correos masivos, documentos institucionales y configuración de agentes AI. Los agentes están orientados a la atención de consultas públicas, apoyo al proceso de inscripción, búsqueda documental controlada y soporte operativo para el comité organizador.

## English

COMAM Portal AI is an institutional web platform for the American Masonic Conference — COMAM. The project includes a public portal, content management, institutional history, articles, documents, statutes, participating lodges and obediences, as well as a registration module for the COMAM 2026 Conference in Santiago, Chile.

The solution includes a secure administrative backoffice with OTP-based two-factor authentication, user management, registration management, guest administration, mass email communications, institutional document management, and AI agent configuration. The AI agents are designed to support public inquiries, assist with the registration process, provide controlled document retrieval, and support the organizing committee’s operations.
